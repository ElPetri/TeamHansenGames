The 5-Step Process

  1. Build pages and scripts in VS Code.
  2. Use Copilot and other AI models to write code, fix bugs, and explain errors.
  3. Save your changes in GitHub.
  4. Deploy with Cloudflare Pages and Worker.
  5. Point the domain to production and share your project.

How AI Helped at Every Step

  • Planned what to build first.
  • Generated starter code and page text.
  • Debugged errors and suggested fixes.
  • Kept everything simple and beginner friendly.

AI was integral from start to finish. I used AI in every phase of this project, with VS Code Copilot as the main coding partner.
For planning, I always started in Planning mode using Opus 4.6 to develop a solid game plan. Once happy with the plan, I switched to Agent mode and used 5.3 Codex for coding and debugging. This workflow helped minimize Premium AI Requests and made the process easier for beginners.
